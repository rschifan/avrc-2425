Example 1: Basic Data-Driven SVG

This example demonstrates the core concept of data binding in D3.js.

Data binding connects your data to visual elements. In this basic example:

  • Each data point creates a circle in the SVG
  • The x position comes from the data's x value
  • The y position comes from the data's y value
  • The circle radius comes from the data's value

No scales or axes are used - data values are mapped directly to pixel positions.
